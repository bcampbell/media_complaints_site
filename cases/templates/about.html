About

This site takes complaints from the Press Complaints Commission, augments them with further data and presents them in a nice easy-to-browse way.

The source code is available here, under the GNU Affero General Public License.

This site is a Media Standards Trust project.
It was built by Ben Campbell, based on an outline by Matthew Cain and with feedback from Martin Moore.
Camilla Schick and Charlotte Hawkins added data to the complaints by hand. They tracked down links to articles, found related complaints and wrote understandable descriptions.

The PCC data is extracted using a scraper hosted on ScraperWiki.
See http://scraperwiki.com/scrapers/pcc-decisions-mark-3/ for code and raw data.

The site was implemented using the Django framework, and uses boilerplate.css
